
# SFLK programming language

```sflk
pr  3  - (1  +  1) # prints 1 #
pr (3  -  1) +  1  # prints 3 #
pr  3  -  1  +  1  # prints 3 #
```

```sflk
uwu < "o"
uwu < uwu + "wo"
pr uwu # prints owo #
```

```sflk
f < {
    x < 2
    pr x + 2 # prints 4 #
}
do f
do {
    pr "uwu" # prints uwu #
}
```

It has
assignment, +, -, strings, integers, print, parenthesis, comments,
code blocks.
And more is comming soon!
